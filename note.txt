Process Flow:
- Check if session exist
  - If not, generate unique session. Check from cache filename
- Get CURL Metadata
- Create / Open unique filecache json
- Write metadata in new line
- Save and close
- Return response success

Download Flow:
- Check if session is exist && File cache with session id exist
  - If not, 403
- Parse and loop cache
- Write in Excel
- Download
